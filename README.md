# 🐳 Docker Microservices Project

## 📌 Overview
This project demonstrates how to run a simple microservice using Docker Compose.

## 🛠️ Tech Stack
- Docker
- Docker Compose
- Node.js

## ⚙️ How It Works
- A Node.js service runs inside a container
- Docker Compose manages multi-container setup

## ▶️ Run Project

```bash
docker-compose up --build
